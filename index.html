About Me

I'm a database developer with report writing background. I've been a database developer for 5+ years, and I have a BS degree in Computer Science from University of Maryland University College. Though I've written simple web pages over the years, there are major gaps in my coding abilities. Last year, I decided to start building more complicated web applications. I discovered I like writing web applications.

Also, anyone in the development space is fully aware that full stack developers are always in great demand. No wonder that full stack web development is one of the most popular developer occupations today.

So I'm in this bootcamp because I want to be a full stack developer. Hopefully after 6 months of writing code intensely, I can take the next steps to slowly transition into a full stack developer role.
